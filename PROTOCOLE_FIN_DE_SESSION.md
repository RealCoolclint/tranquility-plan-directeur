# Protocole — Fin de session Tranquility Suite
## V2.0 — 08 juillet 2026

## Ce que JARVIS produit systématiquement en fin de session

1. Le récap `.md` de la session — texte complet, rédigé dans le chat
2. Le patch Plan Directeur — pas le document entier : décisions à ajouter (format
   tableau identique, point d'insertion précisé), paragraphes narratifs à mettre à
   jour (texte exact avant/après), nouvelles entrées de carnet de pièges
3. Un seul prompt Cursor qui applique le patch + écrit le récap dans le repo réel
